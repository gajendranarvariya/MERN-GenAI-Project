# 🚀 MERN + Gemini AI Career Assistant Platform

An AI-powered full-stack MERN application that helps users analyze resumes, improve job preparation, and generate ATS-friendly resumes using Gemini AI.

The platform allows users to:

- Securely authenticate using JWT
- Analyze resumes and job descriptions
- Generate AI-powered career insights
- Identify skill gaps
- Generate behavioral & technical interview questions
- Create personalized preparation plans
- Generate ATS-friendly resumes
- Export resumes as PDF using Puppeteer
